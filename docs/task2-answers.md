# Требования к функции "Получение списка магазинов при переходе на экран выбора"

1. При отображении экрана с магазинами партнеров отправить на сервер GET-запрос.
2. Сервер должен обработать запрос и вернуть JSON для рендера экрана.
3. В случае, если ни один магазин не найден, загрузить пустой экран с надписью "Нет доступных магазинов"
4. Запрос возвращает JSON со списком магазинов. В каждой строке списка передать атрибуты:
   -  название
   -  логотип
   -  ближайшее окно доставки
   -  внешняя ссылка для перехода по клику
5. Магазины на экране отобразить в алфавитном порядке по наименованию
6. Пример REST API запроса:
```bash
GET
https://petzel.ru/api/v1/stores
'Accept: application/json'
```
7. Пример ответа:
```bash
{

      "name": "METRO",

  ]
}
