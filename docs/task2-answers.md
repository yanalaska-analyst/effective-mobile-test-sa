# Требования к функции "Получение списка магазинов при переходе на экран выбора"

1. При отображении экрана с магазинами партнеров отправить на сервер GET-запрос.
2. Сервер должен обработать запрос и вернуть JSON для рендера экрана.
3. Запрос возвращает JSON со списком магазинов. В каждой строке списка передать атрибуты:
   -  название
   -  логотип
   -  ближайшее окно доставки
   -  внешняя ссылка для перехода по клику
4. Магазины на экране отобразить в алфавитном порядке по наименованию

## Пример REST API запроса:
```bash
GET
https://petzel.ru/api/v1/stores
Accept: application/json
```
## Пример ответа:
```bash
{
  "stores": [
    {
      "name": "METRO",
      "logo": "https://petzel.ru/images/logos/metro.png",
      "nearest_delivery_window": {
        "start": "2026-01-06T21:00:00+03:00",
        "end": "2026-01-06T23:00:00+03:00"
      },
      "external_link": "https://metro.ru"
    },
    {
      "name": "Ашан",
      "logo": "https://petzel.ru/images/logos/auchan.png",
      "nearest_delivery_window": {
        "start": "2026-01-06T21:00:00+03:00",
        "end": "2026-01-06T23:00:00+03:00"
      },
      "external_link": "https://auchan.ru"
    }
  ]
}
```
## Примеры ошибок:
Код 404
```bash
{
  "error": {
    "code": "STORES_NOT_FOUND",
    "message": "Нет доступных магазинов для выбранной локации"
  }
}
```
Код 503
```bash
{
  "error": {
    "code": "SERVICE_UNAVAILABLE",
    "message": "Сервис магазинов временно недоступен. Повторите попытку позже"
  }
}
